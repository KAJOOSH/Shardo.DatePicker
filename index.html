Shardo Date Picker

نمونه‌های کاربردی برای تاریخ شمسی، خروجی میلادی hidden، رنج، زمان، تم Bootstrap، حالت inline و تنظیمات سراسری.

نمونه تنظیمات با Attribute

روی هر فیلد کلیک کنید؛ خروجی شمسی برای کاربر و خروجی میلادی برای سرور در پنل سمت چپ نمایش داده می‌شود.

تاریخ ساده با بستن خودکار

برای فیلدهای معمولی فرم؛ بعد از انتخاب روز بسته می‌شود و مقدار میلادی در hidden input می‌نشیند.

showTime=false autoClose=true hiddenInput

تاریخ و ساعت با تایید

وقتی ساعت فعال است، انتخاب نهایی با دکمه تایید انجام می‌شود و دقیقه‌ها بر اساس step تنظیم می‌شوند.

showTime=true timeStep=5 selectedText

رنج تاریخ

تاریخ شروع و پایان را انتخاب کنید؛ هنگام حرکت موس، محدوده به صورت پیش‌نمایش دیده می‌شود.

range=true hover preview custom separator

محدودیت، تعطیلی و مقدار اولیه

ورودی اولیه می‌تواند شمسی یا میلادی باشد؛ جمعه‌ها و تاریخ‌های تعطیل با ظاهر جدا نمایش داده می‌شوند.

value=gregorian min/max holidays

JSON Attribute

برای تنظیمات طولانی می‌توانید بخشی از optionها را در خود data-shardo-datepicker به صورت JSON بگذارید.

JSON config weekStart yearRange

جای‌گذاری داینامیک پنجره

placement روی auto قرار دارد؛ کنترل بر اساس فضای صفحه بهترین جهت را انتخاب می‌کند.

placement=auto responsive sheet

همه تنظیمات رایج در یک فیلد

این نمونه ترکیبی از مقدار اولیه، hidden input، تم Bootstrap، محدودیت تاریخ، تعطیلی، فرمت انتخاب و بستن خودکار است.

bootstrapTheme theme=auto hiddenInputId disabledDates inputReadonly=false

خروجی آخرین انتخاب

روی یکی از نمونه‌ها کلیک کنید.

راهنمای سریع

  • نمایش فیلد برای کاربر شمسی است، اما hidden input مقدار میلادی را برای سرور نگه می‌دارد.
  • برای تاریخ‌های ورودی مثل value، min، max و holidays می‌توانید شمسی یا میلادی وارد کنید.
  • در حالت range، بعد از انتخاب تاریخ دوم مقدار نهایی ثبت می‌شود.
  • در حالت time، دکمه تایید فقط وقتی لازم باشد نمایش داده می‌شود.
  • دکمه تنظیمات سراسری پایین، optionهای عمومی را روی همه کنترل‌های موجود اعمال می‌کند.

نمونه inline

این حالت همیشه داخل صفحه است و دکمه‌های غیرکاربردی مثل بستن نمایش داده نمی‌شوند.
